# ☕ Premium Cafe Website

A modern, responsive, and visually appealing cafe website built with **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

Inspired by premium coffee shop and restaurant websites, this project delivers an elegant user experience with smooth animations, responsive layouts, menu filtering, gallery browsing, contact forms, and dark/light theme support.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/AnmolAnubhav07/premium-cafe-website.git
```

Navigate to the project directory:

```bash
cd premium-cafe-website
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

The application will run at:

```plaintext
http://localhost:5173
```

---

## 🏗️ Build for Production

Generate a production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
